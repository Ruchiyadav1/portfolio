E-GURU LANDING PAGE

The "E-Guru" landing page is a small-scale web project aimed at promoting an online learning platform. The platform, titled "E-Guru," offers a convenient solution for remote learning with access to expert guidance from the comfort of one's home.

TECHNOLGY USED:

HTML: The structure and content of the web page are built using HTML (Hypertext Markup Language), providing the foundation for presenting information.
CSS with Tailwind Framework: Tailwind CSS, a utility-first CSS framework, is employed for styling the HTML elements, ensuring a responsive and visually appealing design.

Project Goals:

Promotion: The primary goal of the E-Guru landing page is to effectively promote the online learning platform and attract potential users.
